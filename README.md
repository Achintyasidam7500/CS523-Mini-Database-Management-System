## 📊 How the B-Tree Index Helps
A B-Tree is a data structure that keeps data in sorted order. It helps the app find records faster than a plain list.

In simple terms:
- The app does not scan every item one by one
- It can move through data in steps
- Search works well even when the data grows
- The system stays easy to use
